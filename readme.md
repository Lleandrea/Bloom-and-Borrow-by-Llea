## Project Purpose
This project is a website for "Bloom & Borrow by Llea," an eternal and faux flower rental business. The website showcases various floral collections and presentation styles available for weddings, events, and special occasions. The purpose is to provide potential customers with information about the rental services, display the available collections, and provide contact information for bookings.

## Project Structure

### Files Included:
1. **index.html** - The main HTML file containing the website structure and content
2. **styles.css** - The CSS stylesheet that provides styling and visual design for the website
3. **README.md** - This documentation file

### HTML Structure (index.html)
The HTML file contains:
- A main heading with the business name
- An introductory paragraph welcoming visitors
- Six floral collections: Blue Bliss, Forest Feast, Wildest Dreams, Ivory Magic, Pretty Pink, and Cocoa Bliss
- Four presentation style options: Silver Ice Buckets, Wooden Surface Level Boxes, Flower Table Runners, and Elegant Vases
- A "Why Choose Us?" section with benefits listed
- A contact section with the business email address

### CSS Structure (styles.css)
The CSS file includes styling for:
- Body background and general font styling
- Main heading with ID selector (#main-heading)
- Intro paragraph with class selector (.intro)
- Collection containers with elegant backgrounds
- Presentation style containers
- Contact section styling
- Responsive image styling
- Typography and spacing throughout

## How to View the Webpage

### Method 1: Open Directly in Browser
1. Ensure all three files (index.html, styles.css, and README.md) are in the same folder
2. Locate the **index.html** file on your computer
3. Double-click the **index.html** file
4. The webpage will open in your default web browser

### Method 2: Using a Text Editor
1. Open the folder containing your project files
2. Right-click on **index.html**
3. Select "Open with" and choose your preferred web browser (Chrome, Firefox, Edge, Safari, etc.)

### Important Notes:
- Make sure **styles.css** is in the same folder as **index.html** for the styling to work properly
- The files must maintain their exact names: `index.html` and `styles.css`
- No internet connection is required to view the webpage locally
